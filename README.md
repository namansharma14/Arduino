# Arduino

## Crown Currency — Competitor Intelligence Tool

A live competitor-rate dashboard for Crown Currency lives in
[`competitor-intel/`](./competitor-intel/) — scraping competitor websites, capturing
counter-heard rates and free-text intel, and building rate trends with an auto-generated
insights panel. See [`competitor-intel/README.md`](./competitor-intel/README.md) to run it.

```bash
cd competitor-intel && npm install && npm run seed && npm start   # → http://localhost:4000
```
